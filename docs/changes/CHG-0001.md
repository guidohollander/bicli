# CHG-0001 Global Dev Command And Windows Executable

## Scope

Add packaging and installation scripts so the validator can be:

- exposed globally during development without rebuilding for every source change
- packaged as a Windows executable

## Implemented delta

- Added `bicli` wrapper command for the compiled build.
- Added `bicli-dev` wrapper command that runs `src/index.ts` through `tsx`.
- Added `scripts/install-global-dev.ps1` to install and globally link the commands.
- Added `scripts/build-win-exe.ps1` to build `release/bicli.exe`.
