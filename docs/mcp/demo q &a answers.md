# Demo Q&A Answers

This file intentionally contains only generic examples suitable for a public repository.

## Example Repository

- Repository: `sample_beinformed_repo`
- Purpose: demonstration of repository-grounded MCP questions

## Example Questions

1. `Explain the repository architecture`
2. `Which files define the case model?`
3. `Describe the interaction patterns used by the portal`
4. `Trace the links for the sample portal tab`
5. `Describe the _Case workflow pattern in project SC Sample`

## Example Answer Shape

Repository-grounded answers should usually include:

- a short synthesis
- the strongest matching files
- a note that the synthesis is inferred from repository evidence
- optional repository-model context when the question is architectural or modeling-heavy

## Public-Safe Guidance

- Do not check in real customer repository names.
- Do not check in local user profile paths.
- Do not check in customer-specific business analysis or repository content extracts.
